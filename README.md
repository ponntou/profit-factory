# obsidian

AIアシスタント「アンチグラビティ」とユーザーによる、ナレッジ蓄積用リポジトリです。

## 同期について（新方式）
このフォルダはGitHubを介して同期されます。
1. **AI（アンチグラビティ）**: ファイルを保存するたびに、このリポジトリ（`main`ブランチ）へ自動的にプッシュします。
2. **デスクトップ（Windows）**: GitHub Desktopにより、GitHubから自動的にプル（ダウンロード）します。
3. **スマホ**: Windows側のDropboxが変更を検知し、スマホのObsidianへ同期します。

## 運用ルール
1. **成果物/**: プロジェクトごとのコードや設計図を保存。
2. **質疑応答/**: 対話記録をMarkdown形式で保存。
3. **自動同期**: AI側の保存は即座にプッシュされます。
