「資格は取ったけど、実務に活きていない」  
「勉強した内容をすぐ忘れてしまう」

そんな悩みを解決するために、私は **Obsidian × 生成AI** を  
組み合わせた学習環境を構築しました。

この記事では、

- 知識を“蓄積”できる環境構築

---

## 1\. 環境構築

## 1-1. 全体構成イメージ

PCとスマホをDropboxで同期し、Obsidianで同一Vaultを扱います。

![](https://storage.googleapis.com/zenn-user-upload/4b42bbafbb6f-20260228.png)

- PC：執筆・体系化
- スマホ：閲覧・スキマ復習
- Dropbox：Vault同期

※iPhoneの場合はDropboxをiCloudに置き換えた方がよりシンプルな構成となります。

---

## 1-2. Dropbox、Dropsync、Obsidianのインストール

### ■ PC版

事前にDropboxをインストールしておく

1. Obsidianを公式サイトからダウンロード
2. Dropbox配下にVaultを作成  
	※Vaultは保管庫。Obsidianのローカルの専用フォルダになります。

例：  
Dropbox/  
└─ ObsidianVault/

### ■ スマホ版（Android/iOS）

事前にDropbox、Dropsyncをインストールしておく  
DropsyncでObsidianVaultと同期し、スマホのローカルフォルダにダウンロードしておく。

1. Obsidianアプリをインストール
2. ローカルフォルダにDropbox同期フォルダを指定

これでPCとスマホが同じノートを参照できます。

---

## 1-3. プラグイン

Obsidianのプラグインをインストールしておきます。  
プラグインの有効化も忘れずに！  
インストール方法はググるか生成AIに聞けば解決すると思いますので割愛します💦

### Dataview

ノートを“データベース化”できます。（必須）

### Folder Notes

フォルダ自体に説明が記載できるようになります。（任意）  
体系化に便利です。

例. 04\_テスト設計技法フォルダの説明

```
├─ 04_テスト設計技法/
│  ├─ 同値分割.md
│  ├─ 境界値分析.md
│  ├─ デシジョンテーブル.md
│  ├─ 状態遷移テスト.md
│  └─ 経験ベーステスト.md
```

## 1-4. DropboxでPC×スマホ連携

ポイントは：

- VaultをDropbox配下に置く
- スマホでも同じフォルダを指定

これだけでクラウド同期が可能です。

## 2-2. テーマ毎ノート作成（生成AI活用）

手順

1. フォルダ構成案のテーマごとに生成AIに依頼し、ノートの執筆を依頼する
2. Obsidianに貼る
3. 自分の言葉で追記

例. 生成AIに依頼するプロンプトの例

```
テスト計画についてノートにまとめてください。
出来るだけ実務で活きるポイントを記載してください。
それからObsidianに貼り付けできるようにマークダウン形式でお願いします。
```

作成されたノート

```
# テスト計画（Test Planning）

## 1. テスト計画とは
テスト計画とは、

> **テストの目的・方針・範囲・進め方を定め、関係者で合意する活動**

テストを「場当たり的な作業」にしないための、
**テスト活動の設計図**にあたる。

---

## 2. テスト計画の目的

- テストの目的を明確にする
- 何をテストし、何をテストしないかを決める
- リスクに基づいて優先順位を付ける
- 体制・スケジュール・コストを見積もる
- 関係者間の認識を揃える

👉 **テスト計画は「管理」のためだけでなく、「合意形成」のためにある**

---

## 3. テスト計画の位置づけ（JSTQB）

- テストプロセスの最初の活動
- テスト実行前だけでなく、**プロジェクト全体を通して更新される**
- モニタリング・コントロールと密接に関係する

～以下、省略～
```

## 3\. 復習ダッシュボードの作成

ノートにまとめて満足してはいけません。  
知識を定着するためには復習は欠かせません。  
科学的にも記憶が定着しやすいタイミングで復習出来るよう  
復習ダッシュボードを作成します。

## 3-1. 忘却曲線を意識する

エビングハウスの忘却曲線では、

- 1日後
- 3日後
- 7日後
- 30日後

の復習が効果的とされています。

## 3-2. Dataviewで復習一覧表示

Obsidianでノートを新規作成し、以下の内容を記載します。

---

## ① 1日後（超短期記憶）

```
\`\`\`dataview
LIST
FROM "/"
WHERE file.mtime >= (date(today) - dur(1 days))
  AND file.mtime <  (date(today) - dur(0 days))
SORT file.mtime DESC
\`\`\`
```

## ② 3日後（短期 → 中期）

```
\`\`\`dataview
LIST
FROM "/"
WHERE file.mtime >= (date(today) - dur(3 days))
  AND file.mtime <  (date(today) - dur(2 days))
SORT file.mtime DESC
\`\`\`
```

## ③ 7日後（定着チェック）

```
\`\`\`dataview
LIST
FROM "/"
WHERE file.mtime >= (date(today) - dur(7 days))
  AND file.mtime <  (date(today) - dur(6 days))
SORT file.mtime DESC
\`\`\`
```

## ④ 30日後（長期記憶化）

```
\`\`\`dataview
LIST
FROM "/"
WHERE file.mtime >= (date(today) - dur(30 days))
  AND file.mtime <  (date(today) - dur(29 days))
SORT file.mtime DESC
\`\`\`
```

上記のように記載すると、1日前、3日前、7日前、30日前に執筆した  
ノートが一覧で表示されるようになります。
