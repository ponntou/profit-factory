# アンチグラビティ作業フォルダへようこそ

このフォルダは、AIアシスタント「アンチグラビティ」が作成した成果物や、ユーザー様とのやり取りを蓄積するための場所です。

## フォルダ構成
- `成果物/`: プロジェクトごとのコード、設計ドキュメント、画像などの完成品を保存します。
- `質疑応答/`: 日々の質問と回答をMarkdown形式（`.md`）で記録します。Obsidianで閲覧・検索が可能です。

## 同期について（新方式）
このフォルダはGitHubを介して同期されます。
1. **AI（アンチグラビティ）**: ファイルを保存するたびに、このリポジトリ（`main`ブランチ）へ自動的にプッシュします。
2. **デスクトップ（Windows）**: GitプラグインやGitHub Desktopにより、GitHubから自動的にプル（ダウンロード）します。
3. **スマホ**: Windows側のDropboxが変更を検知し、スマホのObsidianへ同期します。

## 運用ルール
1. 成果物が完成するたびに、適切なサブフォルダを `成果物/` 内に作成して保存します。
2. 重要な対話や技術的なＱ＆Ａは `質疑応答/` 内にドキュメント化します。
3. **自動同期**: 保存されたドキュメントは、AIによって即座に GitHub へ同期されます。
