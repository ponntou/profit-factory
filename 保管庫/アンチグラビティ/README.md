# アンチグラビティ作業フォルダへようこそ

このフォルダは、AIアシスタント「アンチグラビティ」が作成した成果物や、ユーザー様とのやり取りを蓄積するための場所です。

## フォルダ構成
- `保管庫/アンチグラビティ/`: AIによる書き込み専用エリア。
    - `[プロジェクト名]/年/月/日/`: `repos/`フォルダのルールに基づき、プロジェクトごとに整理して保存します。
- `保管庫/その他（雑多なもの）`: ユーザーによる自由な保存エリア。AIはここも「読み込み」の対象とします。

## 同期とバージョン管理
このリポジトリは GitHub を介して同期・バックアップされます。
1. **自動プッシュ**: AIがファイルを保存する際、リポジトリ全体の変更を検知して GitHub へ送信（プッシュ）します。
2. **履歴（バージョン）保存**: GitHubに送信されるたびに「保存記録」が残ります。過去のどの時点のファイル内容にも戻ることが可能です。
3. **デバイス間同期**: GitHub ⇄ デスクトップ ⇄ スマホ（Dropbox）の順で自動的に反映されます。

## 運用ルール
1. AIは「保管庫」全域の情報を参照して回答します。
2. 書き込みは常に `保管庫/アンチグラビティ/` 内に行い、履歴を明確にします。
3. **自動同期**: 保存のたびに即座に GitHub へ同期されます。
