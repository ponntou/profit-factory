# 要約：ObsidianをAndroidスマホから無料で使い続ける方法｜おがくずにゃんこ

## 概要
Android端末において、Obsidian無料版ではGoogle Drive等を公式の同期先に選べない制限を回避し、無料でクラウド同期やWebクリップ機能を実現する具体的な運用方法を解説した記事。

## 主要なポイント
- **無料でのマルチデバイス同期**:
  - Androidアプリ単体には同期機能がないため、外部アプリ「Autosync for Google Drive」を利用。
  - ローカルストレージ内のObsidianのVaultフォルダと、Google Drive上の特定のフォルダを定期的に自動同期させることで、あたかもマルチデバイス対応しているかのように運用できる。
- **AndroidでのWeb記事クリップの手法**:
  - Chrome単独では機能不足なため、Firefoxブラウザの拡張機能「Obsidian Web Clipper」を活用。
  - Chromeで共有 → Firefoxで開く → 拡張機能「Obsidian Web Clipper」からワンタップでObsidianのノートに変換・追加するという二段階の連携フローを確立。
