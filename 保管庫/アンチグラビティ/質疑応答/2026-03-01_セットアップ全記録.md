## 2. 計画 (Implementation Plan) の要約
- **目的**: 成果物と質疑応答を自動保存し、Dropbox経由でスマホと同期する。
- **課題**: コンテナからホスト側のパスに直接アクセスできない。
- **解決策**: リポジトリ内に保管庫を置くことで、Git管理とファイルアクセスを両立。

## 3. GitHub連携のトラブルシューティング
- **問題**: `Bad credentials` エラーで接続不可。
- **原因**: `mcp_config.json` に設定されていた古いトークンが失効していた。
- **解決方法**: 新しいパーソナルアクセストークン（Fine-grained）を発行し、設定ファイルを更新。VS Code の再起動により設定を反映。

## 4. 最終的な同期の仕組み
- **ローカル**: 私が「保管庫/アンチグラビティ」に書き込む。
- **GitHub**: `mcp_github_push_files` ツールを使用して、私が自動的にプッシュ。
- **スマホ**: Windows側のDropboxがローカルの変更を検知し、スマホのObsidianへ同期。
